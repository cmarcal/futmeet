# Design Documentation

This folder contains design specifications and prompts for the FutMeet application.

## Contents

### `LOVABLE_DESIGN_PROMPT.md`
Comprehensive design prompt for Lovable.app that includes:

- **Design System**: Complete color palette, typography, spacing, shadows, and responsive breakpoints
- **Screen Specifications**: Detailed visual specifications for all three screens:
  - HomePage (landing page with hero and features)
  - GamePage (player management interface)
  - ResultsPage (team results display)
- **Component Library**: Visual specifications for buttons, inputs, cards, and badges
- **User Flows**: Visual flow diagrams
- **Figma Export Specifications**: Artboard sizes, design token organization, and export requirements

**Purpose**: Use this prompt in Lovable.app or other design tools to generate visual designs and screen mockups for FutMeet. The prompt focuses purely on visual design specifications without technical implementation details.

## Usage

This design prompt is intended to be used with:
- Lovable.app for generating application designs
- Figma for creating detailed design mockups
- Other design tools that accept text-based design prompts

The prompt provides all necessary visual specifications, color codes, typography, spacing, and layout structures needed to recreate the FutMeet design.
